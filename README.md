# Aetheria: Reign of Magic

## Descripción
Simulación estratégica de ecosistema mágico con 5 sistemas interconectados.

## Instalación
1. Descargar la última release desde
2. Extraer archivos
3. Ejecutar .exe (Windows)

## Controles
- **Movimiento Cámara:** WASD / Flechas
- **Pausa:** ESC
- **Panel Controles:** TAB
- **Reiniciar:** R
